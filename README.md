# 英语词汇复习助手

个人免费版微信小程序原型，用测试驱动英语词汇复习。第一版先做本地存储，不依赖服务器或云开发。

## 如何预览

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹：`/Users/kingopc/Desktop/edu`。
4. AppID 可以先选择测试号或使用游客模式。
5. 编译后从“今日”页开始体验。

## 已实现

- 三档难度：首页提供五年级、高中、四六级三个挑战入口。
- 四选一测试：每轮 10 题，看英文单词，从四个中文释义中选择正确答案。
- 错题复测：答错自动进入错题本，可单独复测。
- 学习进度：根据答题结果安排后续复习。
- 间隔复习：根据反馈安排下次复习。
- 错词本：答错单词自动进入错词本。
- 词库选择：五年级、高中、四六级可开关。
- 学习统计：显示当前词库、已学习、已掌握、正确率。
- 我的页面：提供隐私政策、用户协议和清空学习记录。

## 词库位置

样例词库在 `utils/words.js`。后续扩充词库时，保持每个单词包含这些字段：

- `id`
- `text`
- `phonetic`
- `level`
- `book`
- `partOfSpeech`
- `meaning`
- `example`
- `collocation`
- `note`

也可以先按 `data/word-template.csv` 的格式整理 CSV，然后运行：

```bash
node scripts/csv-to-words.js data/word-template.csv utils/words.generated.js
```

检查生成结果无误后，再把内容合并进 `utils/words.js`。

词库数据来源和许可说明见 `docs/vocabulary-sources.md`。

## 云开发配置

当前项目已预留微信云开发能力。词库存放在本地，学习进度和测试记录登录后同步到云数据库。

需要在微信开发者工具里完成：

1. 开通云开发，复制环境 ID。
2. 将 `app.js` 里的 `YOUR_CLOUD_ENV_ID` 替换为真实环境 ID。
3. 创建数据库集合：`users`、`user_progress`、`quiz_logs`。
4. 上传并部署云函数：`login`、`syncProgress`、`saveQuizLog`。
5. 在“我的”页点击“登录并保存进度”测试同步。
