# 英语词汇复习助手

## 定位

个人免费版微信小程序，用测试驱动英语词汇复习。第一版覆盖五年级、高中、四六级三个难度，适合不同基础的学生从选择题开始练习。

## 第一版功能

- 三档难度：首页提供五年级、高中、四六级挑战入口。
- 四选一测试：看英文单词，从四个中文释义中选择正确答案。
- 错题复测：答错的词自动进入错题本，可单独复测。
- 间隔复习：根据反馈安排下次复习时间。
- 错词本：答错的词自动进入错词本，并在后续任务中优先出现。
- 词库选择：五年级、高中、四六级可开关。
- 学习统计：展示当前词库、已学习、已掌握、正确率。

## 后续扩展

- 扩充完整高中、四级、六级词库。
- 增加闯关解锁、限时挑战、连胜反馈、例句填空、易混词辨析。
- 接入微信云开发，支持多设备同步。
