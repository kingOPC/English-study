# 词库数据来源

本项目的小学、高中和四六级扩展词条参考并整理自 ECDICT 英中词典库。其中小学新增词优先按 `zk` 基础词汇标签和常用频率筛选，高中新增词优先按 `gk` 高考词汇标签和常用频率筛选。

- 项目地址：https://github.com/skywind3000/ECDICT
